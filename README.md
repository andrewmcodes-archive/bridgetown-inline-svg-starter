> A simple starter template for building a static Bridgetown site with inlined SVGs using the [bridgetown-inline-svg](https://github.com/andrewmcodes/bridgetown-inline-svg) plugin.

## Install

```sh
bundle install && yarn install
```

## Usage

```sh
yarn start
```
